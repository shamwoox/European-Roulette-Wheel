PROJECT TITLE: Roulette Wheel
PURPOSE OF PROJECT: To simulate a European Roulette Wheel
VERSION or DATE: 3/11/16 v.1
HOW TO START THIS PROJECT: Create a roulette wheel object that will determine if the number is black, red, or green.
Create a main class that uses the roulette wheel object and to calculate the number of reds and blacks, the number of occurances of two same consecutive numbers,
the number of occurances for 3 same consecutive numbers, the longest odd streak and the longest even streak. 
AUTHORS: Paul Vu, Caroline Floom, and Jeffrey Maldonado
USER INSTRUCTIONS: Input the number of spins for three trials.
